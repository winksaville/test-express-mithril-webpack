# Test express mithril webpack

## Prerequesites
- node
- yarn or npm

## Install client and server modules using yarn or npm
```
yarn install
```

## Build client and server
```
yarn build
```

## Start the server
```
yarn start:server
```

# Access http://localhost:3000 or:
```
yarn start:client
```
And you'll see on the web page
```
Hi there
```
